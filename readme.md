# llmc (Rust Edition)

`llmc` 是一个高性能、轻量级的 Git Commit 助手。它能自动分析暂存区变更，并通过 LLM 生成符合规范的提交信息。

## 使用方式

``` bash
llmc
```

输入llmc后，程序会在本地自动查找是否存在Git仓库，如果存在则查看已经add的改动，并将改动发送到大模型获取commit msg，随后展示给用户，用户确认后可以选择提交更改。

``` bash
llmc --model xxx
```

更改使用的model，llmc支持对接多个大模型，通过输入大模型名称来更改默认使用的模型。

``` bash
llmc --model_list
```

获取所有可用的模型列表。

所有模型相关配置都在项目根目录的config.toml中，如果需要加入新模型则在此文件中作出修改，注意程序运行时不允许修改此文件，因为程序需要读取文件配置初始化模型状态。


```
llmc --limit 1000
```

设置最大的改动信息为1000tokens（默认也是1000tokens），当改动信息过大时会发出警告提示用户。
